# NexoraAI — Premium Real Estate AI Automation Website

A premium, modern, conversion-focused agency website built with React, Tailwind CSS, and Framer Motion.

## Tech Stack

- **React 18** — UI framework
- **TypeScript** — Type safety
- **Tailwind CSS** — Utility-first styling
- **Framer Motion** — Smooth animations
- **Vite** — Build tool
- **Lucide React** — Icons

## Features

- Fully responsive (mobile & desktop)
- Smooth scroll animations with intersection observer
- Glassmorphism design cards
- Premium lavender/dark purple color palette
- WhatsApp floating button
- Contact form with validation-ready structure
- SEO optimized meta tags
- Fast loading with Vite

## Sections

1. **Hero** — Headline, subheadline, CTA buttons, floating dashboard preview
2. **Problem** — Pain points with statistic cards
3. **Process** — 4-step automation workflow (highly visible)
4. **Services** — What NexoraAI does (6 service cards)
5. **Benefits** — Why choose NexoraAI (6 benefit cards)
6. **Offer** — WhatsApp automation focus with chat mockup
7. **Trust** — Testimonials, metrics, dashboard preview
8. **CTA** — Contact form with consultation booking
9. **Footer** — Links, contact info, social

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Deployment

Build the project and deploy the `dist` folder to your hosting provider (e.g., Vercel, Netlify, or your own server for nexoraia.store).
